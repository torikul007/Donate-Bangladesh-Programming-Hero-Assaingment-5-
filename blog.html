What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for HTML documents. It represents the page as a structured tree of objects (nodes), where each element, attribute, or text is a node. How do you select an element from the DOM?

To select an element from the DOM in JavaScript, you can use various methods provided by the document object. Here are some commonly used methods:
1. getElementById()
Selects an element by its ID.
2. getElementsByClassName()
Selects elements by their class name (returns an HTMLCollection).
3. getElementsByTagName()
Selects elements by their tag name (e.g., div,h, p,etc.).
4. querySelector()
Selects the first element that matches a CSS selector.

What is the event delegation in the context of DOM, and why is it useful ?

Event delegation is a technique in the context of the DOM where you leverage event bubbling to handle events more efficiently. Instead of adding event listeners to multiple child elements, you add a single event listener to a common parent element. The parent element catches events triggered by the child elements due to event propagation (bubbling up the DOM tree).
How Event Delegation Works:
When an event is triggered on a child element, it first triggers on the child and then bubbles up to its parent, continuing up to the root (document). By attaching the event listener to the parent element, you can capture the event and handle it for any of the child elements.
Why Event Delegation Is Useful:

  • Efficiency: Reduces the number of event listeners, improving performance.
Handles Dynamic Elements: Automatically applies to new child elements added later. Cleaner Code: Centralizes event handling logic

How do you manipulate an event's attributes and styles using DOM?

We can manipulate an event's attributes and styles by accessing the event object, which provides information about the event and its target element. You can change styles like background color or font size directly on the element that triggered the event. Additionally, you can modify attributes like text or disabling a button, and control the event's behavior using methods like preventDefault() to stop default actions.
